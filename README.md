# RL Trading Bot для BTC/USDT

Этот проект — RL-торговый бот на базе Stable-Baselines3 (PPO), интегрирующий данные Bybit, анализ настроения новостей (VADER) и асинхронный live-трейдинг.

## Функции

- RL-среда с индикаторами (RSI, MACD).
- Анализ настроения новостей для улучшения предсказаний.
- Backtesting с backtrader.
- Live-трейдинг с Redis-кэшем.
- Запуск в Docker для изоляции.

## Установка и запуск

1. Клонируйте репозиторий.
2. Установите Docker и Docker Compose.
3. Скопируйте `.env.example` в `.env` и заполните ключи.
4. Скачайте исторические данные в `data/historical_btc.csv` (OHLCV формат).
5. Запустите: `docker-compose up --build`.
6. Тестируйте скрипты внутри контейнера: `docker-compose exec app python scripts/train_model.py`.

## Риски

Это симуляция. Не используйте для реальных сделок без тестирования. Потеря денег возможна.

## Структура

- `src/`: Модули (среда, анализатор).
- `scripts/`: Скрипты (обучение, трейдинг, backtest).
- `models/`: Сохранённые модели.
- `data/`: Исторические данные.
